## git branch

- branch: 가지
- git branch -> * main (default branch)
- branch 분기점: commit 이력 기준
- git branch [이름]
- git checkout || git switch: branch를 바꿈
- branch 지우기: git branch -D [브랜치명]

***

## git merge

- merge: 해당하는 파일에서 다른 파일을 '당겨옴'

- git flow (가장 많이 쓰임)
